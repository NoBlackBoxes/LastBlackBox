# course : scripts : materials
Generate BOM and packing lists

### Materials Format
Name, Slug, Description, Quantity, Datasheet (optional), Supplier (optional), Package, x(mm), y(mm), z(mm), Unit Price (£), Bulk Price (£), Available (new), Available, (used)
